## Overview

This project contains Jupyter notebooks that generate all figures included in this repository.

## Dataset

The analyses use the 1-second interval dataset, which can be accessed here:
http://205.174.165.80/IOTDataset/Datasense/Dataset/dataset/processed_files/

## Usage

Download the dataset from the link above.

Place the dataset in the appropriate directory referenced in the notebooks.

Run the notebooks to reproduce all figures.
